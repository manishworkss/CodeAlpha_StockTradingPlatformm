# Stock Trading Platform

Java console application for stock market simulation with real-time trading.

**CodeAlpha Internship Project - Task 2**

## Features
- Buy/Sell stocks
- Portfolio tracking  
- Real-time prices
- File I/O persistence

## How to Run
1. Download ZIP or clone repository
2. Compile: `javac *.java`  
3. Run: `java TradingPlatform`

## Tech Stack
Java | OOP | File I/O | HashMap
